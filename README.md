# Lumière Studio – Beauty Salon Website UI

## 📌 Project Overview
This project is a UI/UX design of a Beauty Salon website created using Figma.  
The goal is to design a high-conversion website for a local service business.

## 🎯 Objective
- Clearly present services
- Build trust with users
- Improve lead generation
- Encourage booking actions

## 🖥️ Pages Designed
- Homepage
- Services Page
- Contact / Booking Page

## ✨ Key Features
- Clean and modern UI
- Mobile responsive design
- Service pricing visibility
- Testimonials for trust
- Strong Call-to-Action (Book Now)
- Simple booking form

## 🧠 UX Decisions
- Clear hero section with CTA
- Visible pricing for transparency
- Minimal form fields for easy booking
- Mobile-first layout
- Consistent spacing and hierarchy

## 📈 Outcome
This design improves user experience and helps convert visitors into customers.
